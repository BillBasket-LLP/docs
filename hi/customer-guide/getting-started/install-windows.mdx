---
title: "Windows पर इंस्टॉल करें"
description: "अपने Windows कंप्यूटर पर BillBasket डाउनलोड और इंस्टॉल करें।"
---

<Steps>
  <Step title="इंस्टॉलर डाउनलोड करें">
    अपने BillBasket पार्टनर द्वारा दिए गए लिंक या आपके साथ शेयर किए गए डाउनलोड
    पेज से BillBasket का लेटेस्ट इंस्टॉलर डाउनलोड करें।
  </Step>
  <Step title="इंस्टॉलर चलाएँ">
    डाउनलोड की गई `.exe` फ़ाइल पर डबल-क्लिक करें और स्क्रीन पर दिए गए स्टेप्स फ़ॉलो करें।
    अगर Windows कोई सिक्योरिटी प्रॉम्प्ट दिखाए, तो **More info → Run anyway** चुनें —
    नए इंस्टॉल किए गए ऐप के लिए यह सामान्य है।
  </Step>
  <Step title="BillBasket खोलें">
    इंस्टॉलेशन के बाद BillBasket अपने-आप खुल जाता है और आपके डेस्कटॉप तथा Start
    मेन्यू में एक शॉर्टकट जोड़ देता है।
  </Step>
  <Step title="पहली बार के सेटअप पर आगे बढ़ें">
    अपनी दुकान की प्रोफ़ाइल बनाने के लिए [पहली बार का सेटअप](/hi/customer-guide/getting-started/first-time-setup) पर जाएँ।
  </Step>
</Steps>

<Note>
  BillBasket आपका सारा डेटा आपके अपने कंप्यूटर पर एन्क्रिप्टेड रूप में स्टोर करता है, इसलिए
  यह इंटरनेट के बिना भी काम करता रहता है।
</Note>
