## अपने डिवाइस पर देखें

<Tabs>
  <Tab title="फ़ोन">
    <Frame>
      _स्क्रीनशॉट जल्द ही आ रहा है — Android फ़ोन व्यू।_
    </Frame>
  </Tab>
  <Tab title="टैबलेट">
    <Frame>
      _स्क्रीनशॉट जल्द ही आ रहा है — Android टैबलेट व्यू।_
    </Frame>
  </Tab>
  <Tab title="Windows PC">
    <Frame>
      _स्क्रीनशॉट जल्द ही आ रहा है — Windows डेस्कटॉप व्यू।_
    </Frame>
  </Tab>
</Tabs>

<Steps>
  <Step title="इंस्टॉलर डाउनलोड करें">
    अपने BillBasket पार्टनर से मिले लिंक से या आपके साथ शेयर किए गए डाउनलोड पेज
    से नवीनतम BillBasket इंस्टॉलर डाउनलोड करें।
  </Step>
  <Step title="इंस्टॉलर चलाएँ">
    डाउनलोड की गई `.exe` फ़ाइल पर डबल-क्लिक करें और स्क्रीन पर दिए गए स्टेप्स
    फ़ॉलो करें। अगर Windows सिक्योरिटी प्रॉम्प्ट दिखाए, तो **More info → Run
    anyway** चुनें — किसी नई ऐप के लिए यह सामान्य है।
  </Step>
  <Step title="BillBasket खोलें">
    इंस्टॉलेशन के बाद BillBasket अपने-आप शुरू हो जाता है और आपके डेस्कटॉप और
    Start मेनू में शॉर्टकट जोड़ देता है।
  </Step>
  <Step title="पहली बार के सेटअप पर आगे बढ़ें">
    अपनी दुकान की प्रोफ़ाइल बनाने के लिए [पहली बार का सेटअप](/hi/customer-guide/getting-started/first-time-setup) पर जाएँ।
  </Step>
</Steps>

<Note>
  BillBasket आपका सारा डेटा एन्क्रिप्टेड रूप में आपके अपने कंप्यूटर पर स्टोर
  करता है, इसलिए यह इंटरनेट के बिना भी काम करता रहता है।
</Note>
