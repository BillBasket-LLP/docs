BillBasket for Android runs on a phone or tablet, either as your main
billing device or as a companion to a Windows PC already running BillBasket
at your counter.

<Note>
  BillBasket stores all your data on your own device, encrypted, so billing
  keeps working even without internet. See [Understanding
  offline-first](/customer-guide/getting-started/offline-first).
</Note>

## Play Store vs. APK

Most shops install from the Play Store. If your BillBasket partner has
shared an APK file directly instead (for example, ahead of a Play Store
release), Android will ask you to allow installs from that source the first
time — this is only needed once.

## Permissions BillBasket asks for

On first launch, BillBasket asks for:

- **Camera** — needed if you plan to scan barcodes with your phone instead
  of a hardware scanner.
- **Storage** — needed so BillBasket can save backups to your device.

## Solo phone or paired companion

A phone can run BillBasket entirely on its own, with its own store data, or
it can connect as a companion to a Windows computer already running
BillBasket — sharing the same store data live. See [Using BillBasket on
phone + desktop
together](/customer-guide/settings/companion-and-web-access).

## Try it: Sharma Kirana Store

<Steps>
  <Step title="Download the app">
    Rajesh Sharma's staff member Suresh installs BillBasket from the Play
    Store on his phone. (If Rajesh's BillBasket partner had shared an APK
    directly instead, Suresh would install that file instead of using the
    Play Store.)
  </Step>
  <Step title="Allow install from unknown sources (APK only)">
    Since Suresh installed from the Play Store this time, Android doesn't ask
    for this — it only appears when installing an APK directly, and only
    needs approving once.
  </Step>
  <Step title="Open BillBasket and grant permissions">
    On first launch, Suresh allows camera access so he can scan barcodes at
    the counter, and storage access so backups can be saved to the phone.
  </Step>
  <Step title="Choose how this phone will be used">
    Suresh sets his phone up as a companion to the Windows PC already running
    BillBasket at **Sharma Kirana Store**'s counter, so both devices share
    the same live store data. See [Using BillBasket on phone + desktop
    together](/customer-guide/settings/companion-and-web-access).
  </Step>
</Steps>
