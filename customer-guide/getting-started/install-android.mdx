<Steps>
  <Step title="Download the app">
    Install BillBasket from the Play Store, or install the APK shared by your
    BillBasket partner if you're installing outside the Play Store.
  </Step>
  <Step title="Allow install from unknown sources (APK only)">
    If installing an APK directly, Android may ask you to allow installs from
    this source. Approve it — this is only needed once.
  </Step>
  <Step title="Open BillBasket and grant permissions">
    On first launch, allow camera access if you plan to scan barcodes, and
    storage access for backups.
  </Step>
  <Step title="Choose how this phone will be used">
    A phone can run BillBasket on its own, or connect as a companion to a
    Windows computer already running BillBasket at your counter. See [Using
    BillBasket on phone + desktop
    together](/customer-guide/data-and-backup/phone-and-desktop-together).
  </Step>
</Steps>
