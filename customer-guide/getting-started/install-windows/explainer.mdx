title: "Install on Windows"
description: "What the BillBasket Windows installer sets up on your PC or laptop, and what to expect the first time it runs."
---

BillBasket for Windows is a single installer that puts everything your
counter needs on one PC or laptop: the billing app, a local encrypted
database, and a desktop shortcut — no separate database or server to set up.

<Note>
  BillBasket stores all your data on your own computer, encrypted, so it
  keeps working even without internet. See [Understanding
  offline-first](/customer-guide/getting-started/offline-first/explainer).
</Note>

## Why a security prompt shows up

The first time you run a new installer, Windows shows a warning because it
doesn't yet recognize the app. Choosing **More info → Run anyway** is
expected and safe for a BillBasket installer you downloaded from your
BillBasket partner or the official download page.

## What happens after install

BillBasket launches automatically once installation finishes, and adds a
shortcut to your desktop and Start menu so you can open it directly next
time. From there, you move straight into [first-time
setup](/customer-guide/getting-started/first-time-setup/explainer) to create your
store profile.
