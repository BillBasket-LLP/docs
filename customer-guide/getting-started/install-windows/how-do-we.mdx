## Try it: Sharma Kirana Store

<Steps>
  <Step title="Download the installer">
    Rajesh Sharma downloads the latest BillBasket installer from the link his
    BillBasket partner shared with him.
  </Step>
  <Step title="Run the installer and approve the security prompt">
    He double-clicks the downloaded `.exe` file. Windows shows a security
    prompt since it's a newly installed app, so Rajesh chooses **More info →
    Run anyway** and follows the on-screen steps.
  </Step>
  <Step title="Open BillBasket">
    BillBasket launches automatically after installation and adds a shortcut
    to his desktop and Start menu for next time.
  </Step>
  <Step title="Continue to first-time setup">
    Rajesh continues straight into [First-time
    setup](/customer-guide/getting-started/first-time-setup/explainer) to create his
    store profile for **Sharma Kirana Store**.
  </Step>
</Steps>
