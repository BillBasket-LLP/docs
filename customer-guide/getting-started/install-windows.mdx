---
title: "Install on Windows"
description: "Download and install BillBasket on your Windows computer."
---

<Steps>
  <Step title="Download the installer">
    Download the latest BillBasket installer from the link your BillBasket
    partner or the download page shared with you.
  </Step>
  <Step title="Run the installer">
    Double-click the downloaded `.exe` file and follow the on-screen steps.
    If Windows shows a security prompt, choose **More info → Run anyway** —
    this is expected for a newly installed app.
  </Step>
  <Step title="Open BillBasket">
    BillBasket launches automatically after installation and adds a shortcut
    to your desktop and Start menu.
  </Step>
  <Step title="Continue to first-time setup">
    Head to [First-time setup](/customer-guide/getting-started/first-time-setup) to create your store profile.
  </Step>
</Steps>

<Note>
  BillBasket stores all your data on your own computer, encrypted, so it
  keeps working even without internet.
</Note>
