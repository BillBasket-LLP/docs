Your Android phone can connect to your Windows BillBasket over your
shop's Wi-Fi, so staff can bill from a phone while everything stays in
sync with the counter machine — no internet required, just the same
local network.

## Understanding the fields

- **Host device** — the Windows PC that holds the main store data and
  runs the local server other devices connect to. There's always exactly
  one host per store.
- **Client device** — a phone (or another device) that pairs to the
  host over Wi-Fi and bills against the same store data, rather than
  keeping its own separate copy.
- **Pairing** — a one-time setup step per client device; once paired, the
  connection is remembered so staff don't need to re-pair every shift, as
  long as the phone is on the same local network as the host.

<Note>
  Because this connects over your shop's own Wi-Fi network, both devices
  need to be on the same network — this isn't an internet-based sync, so
  a client device taken off-site won't be able to reach the host.
</Note>
