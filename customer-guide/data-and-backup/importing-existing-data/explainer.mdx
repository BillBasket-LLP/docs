Import master data (products, customers, suppliers) from an Excel file
under **Settings → Data Import/Export**. Download the template first to
match the expected columns.

## Understanding the fields

- **Import template** — download this first; its column headers are what
  BillBasket expects (name, price, tax rate, opening stock, etc.). Typing
  data into a spreadsheet with different column names or order will fail
  to map correctly.
- **Import file** — your filled-in copy of the template, uploaded back
  in. BillBasket reads every row and creates the matching records in one
  pass.
- **Master vs. transactional data** — this importer is for master data
  (products, customers, suppliers, opening stock) that exists before any
  billing happens. It isn't for importing historical invoices or
  transactions — those are created going forward through normal billing.
